# Описание сервиса

Сервис "Index" предназначен для управления и обработки данных, обеспечивая эффективный доступ и хранение информации. Он
поддерживает различные форматы данных и предоставляет API для взаимодействия с другими системами.
## Основные функции
- Получение текста документа
- Обработка (Chanking) текста на части
- Векторизация текста с использованием моделей Ollama (пример есть так же в `services/index/internal/neuro`)
- Сохранение и извлечение данных из базы данных (Qdrant), пример реализации есть в `services/chat/internal/qdrant-wrapper/qdrant_wrapper.go`
## Технологии
- Langchain: https://pkg.go.dev/github.com/tmc/langchaingo (работа с LLM и цепочками обработки)
- Ollama: https://ollama.com/ (локальное использование моделей LLM)
- Qdrant: https://qdrant.tech/ (векторная база данных)
- NATS: https://nats.io/ (для обмена сообщениями между сервисами)
